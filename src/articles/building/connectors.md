Wiring Loom :
Motor lengths are measured with 40mm out of hole, 125mm past point X/Y

| Motor | Length (mm) | Routing |
| ----- | ----------- | ------- |
| 1     | 475         | RHS     |
| 2     | 350         | RHS     |
| 3     | 230         | RHS     |
| 4     | 320         | LHS     |
| 5     | 430         | LHS     |
| 6     | 550         | LHS     |
| 7     | 580         | RHS     |
